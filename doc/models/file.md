
# File

## Structure

`File`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `size` | `number \| undefined` | Optional | The size of the document in bytes. |
| `filename` | `string \| undefined` | Optional | The original filename of the document. |
| `contentType` | `string \| undefined` | Optional | The MIME content type of the document. |
